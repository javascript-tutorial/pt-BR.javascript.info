# Contar propriedades

Escreva uma função `count(obj)` que retorna o número de propriedades no objeto:

```js
let user = {
  name: 'John',
  age: 30
};

alert( count(user) ); // 2
```

Tente tornar o código o mais curto possível.

P.S. Ignore as propriedades simbólicas e conte apenas as "regulares".
