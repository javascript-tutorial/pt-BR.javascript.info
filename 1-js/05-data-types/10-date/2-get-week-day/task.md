# Mostrar um dia da semana

Escreva uma função `getWeekDay(date)` para mostrar o dia da semana no formato curto: 'MO', 'TU', 'WE', 'TH', 'FR', 'SA', 'SU'.

Por exemplo:

```js no-beautify
let date = new Date(2012, 0, 3);  // 3 Jan 2012
alert( getWeekDay(date) );        // deve mostrar "TU"
```
